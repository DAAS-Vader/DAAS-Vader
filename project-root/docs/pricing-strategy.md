# 💰 DaaS 플랫폼 가격 정책

## 1. 가격 정책 개요

**핵심 철학**: "기존 서비스 대비 60-80% 저렴하면서도 투명하고 예측 가능한 가격"

- **경쟁 대상**: Vercel, Netlify, AWS Lambda, Railway
- **차별화**: 탈중앙화로 인한 저비용 + 크립토 네이티브 결제
- **투명성**: 모든 가격이 스마트 컨트랙트에 기록

## 3. 종량제 가격 (Pay-as-you-go)

### 3.1 함수 실행 비용

```
🔄 실행 시간 기반:
- 기본 단위: 100ms
- GB-초당 비용: $0.0000008 (AWS Lambda의 60% 수준)

예시:
- 128MB 함수, 200ms 실행 → $0.000000032
- 512MB 함수, 1초 실행 → $0.0000004
- 1GB 함수, 5초 실행 → $0.000004
```

## 4. 결제 방식

### 4.1 크립토 결제 (기본)

```
🪙 SUI 토큰:
- 모든 결제는 SUI로 처리
- 스마트 컨트랙트 자동 결제
- 월간 정산: 매월 1일 자동 차감

🔄 토큰 교환:
- 내장 DEX 연동으로 자동 스왑
- 사용자는 원하는 토큰으로 결제 가능
- 백엔드에서 SUI로 자동 변환
```

### 4.2 피아트 결제 (옵션)

```
💳 전통적 결제:
- 신용카드/직불카드 (Stripe 연동)
- 자동으로 SUI 구매 후 결제
- 3% 결제 수수료 추가

🏦 기업 결제:
- 송금 및 청구서 결제
- 월/연간 선결제 할인 (10-20%)
- 맞춤 청구 주기
```

## 8. 구현 로드맵

### Phase 1: 기본 요금제 (MVP)
- 무료 티어 + 프로 티어만 제공
- SUI 토큰 결제만 지원
- 기본 워커노드 풀 운영

### Phase 2: 확장된 요금제
- 팀 티어 추가
- 피아트 결제 옵션
- 지역별 노드 풀 운영

### Phase 3: 엔터프라이즈
- 엔터프라이즈 티어 출시
- 맞춤 계약 및 SLA
- 전담 지원팀 구성

### Phase 4: 고도화
- AI 기반 가격 최적화
- 동적 가격 조정
- 예측 분석 및 추천

이 가격 정책은 경쟁력을 유지하면서도 지속가능한 생태계를 구축하여 모든 참여자(개발자, 노드 운영자, 플랫폼)가 윈-윈할 수 있는 구조를 목표로 합니다.
